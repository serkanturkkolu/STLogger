# STLogger
Logger what can work with any output stream or file for C++

You can use this library on <b>linux</b> , <b>macOS</b> , <b>windows</b> . You can create executable using <b>build.sh</b> in gcc(linux and macOS) and mingw(windows).
If you want to use MSVC(windows), you should add STLogger.h and STLogger.cpp to your project. It will work without any problems.
